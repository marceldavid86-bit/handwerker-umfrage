# 🔨 Handwerker-Umfrage 2026

Deine kostenlose Umfrage für Handwerker – in 10 Minuten online!

---

## 🚀 Schritt-für-Schritt Online stellen

### 1. GitHub Account erstellen (kostenlos)
👉 https://github.com → „Sign up"

### 2. Neues Repository erstellen
- Oben rechts auf „+" → „New repository"
- Name: `handwerker-umfrage`
- Public auswählen
- Auf „Create repository" klicken

### 3. Dateien hochladen
- Im neuen Repository auf „uploading an existing file" klicken
- Den kompletten Inhalt dieses Ordners hochladen
- Auf „Commit changes" klicken

### 4. Vercel Account erstellen (kostenlos)
👉 https://vercel.com → „Sign Up" → „Continue with GitHub"

### 5. Projekt deployen
- „New Project" klicken
- Dein Repository `handwerker-umfrage` auswählen
- Auf „Deploy" klicken
- ✅ Fertig! Du bekommst einen Link wie:
  `https://handwerker-umfrage.vercel.app`
